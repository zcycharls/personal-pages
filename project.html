孬孬Naonao

孬孬是一个 Windows 桌面专注陪伴工具。它把一只小树袋熊放在屏幕角落，用比较轻的方式提醒你回到当前任务，而不是再塞给你一个复杂的待办清单。项目入口是 naonao.help。

它主要面向容易走神、需要外部锚点的人，尤其适合 ADHD 朋友使用。孬孬支持番茄钟、任务锚、定时签到、一起专注、想法冰箱、专注数据统计和飞书监督：走神的时候，它负责把你拉回来；突然冒出的想法，可以先放进“冰箱”，等当前任务结束后再处理。

技术上，孬孬是一个开源的 Windows 桌面应用，采用 MIT 许可证。它可以接入云端 AI，也可以按需使用本地模型；不要求注册账号，不做遥测，API Key 保存在本机。这个项目的目标不是制造压力，而是用一个更温和的桌面伙伴，帮人把注意力留在眼前的一件事上。

真实之镜 Real Mirror

真实之镜是一个 Android “真实镜子”应用。它会打开手机前置摄像头做实时预览，并把前置摄像头默认的镜像显示再水平翻转一次，让屏幕里看到的自己更接近别人眼中的方向。

这个应用用 Kotlin、Jetpack Compose 和 CameraX 实现。核心做法不是逐帧处理图像，而是在预览层对 CameraX 的 PreviewView 做水平翻转，所以预览延迟基本保持为系统相机预览的延迟。应用只声明并请求相机权限。

可以直接下载测试版 APK：real-mirror.apk。这是测试签名包，适合手机测试；安装时如果系统提示“未知来源应用”，需要允许当前浏览器或文件管理器安装 APK。
